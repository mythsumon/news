# Design Tokens & Spacing Guide

## Colors
- Background: `#F9FAFB`
- Text Primary: `#0F172A`
- Text Secondary: `#64748B`
- Accent Green: `#16A34A`
- White: `#FFFFFF`
- Gray Borders: `#E5E7EB`, `#E5E7EB` (100)

## Typography
- Font Family: Pretendard / Noto Sans KR (Korean), Noto Sans (EN/MM)
- Headline Weight: 700
- Body Weight: 400-500
- Sizes:
  - Headline Large: 32px (2xl) / 24px (mobile)
  - Headline Medium: 24px (xl) / 20px (mobile)
  - Body: 16px (base) / 14px (mobile)
  - Small: 12px-14px

## Spacing (8px Grid)
- XS: 4px (gap-1)
- SM: 8px (gap-2)
- MD: 16px (gap-4)
- LG: 24px (gap-6)
- XL: 32px (gap-8)
- 2XL: 48px (gap-12)

## Border Radius
- Card: 16px (rounded-card)
- Button: 8px (rounded-lg)
- Pill: 999px (rounded-full)

## Shadows
- Card: `0 1px 3px 0 rgba(0, 0, 0, 0.1), 0 1px 2px 0 rgba(0, 0, 0, 0.06)`
- Card Hover: `0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05)`

## Breakpoints
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

## Component Specifications

### Utility Bar
- Height: 40px (mobile hidden)
- Padding: 8px 32px
- Border: bottom 1px gray-100

### Header
- Height: 64px (mobile: 56px)
- Padding: 16px 32px (mobile: 12px 16px)
- Sticky positioning

### Promo Banner
- Height: 320px (desktop) / 300px (tablet) / 200px (mobile)
- Border Radius: 16px
- Gradient overlay: black/60 to transparent

### Trending Ticker
- Height: 48px
- Background: #0F172A
- Label: #16A34A

### Hero Cards
- Large Card: 16:9 aspect ratio, min-height 400px (mobile) / 500px (desktop)
- Small Cards: 16:9 aspect ratio, height 150px (desktop) / 120px (mobile)
- Hover: translate-y -1px, scale image 1.03x

### Mobile Bottom Tab Bar
- Height: 64px (with safe area)
- Padding: 8px 0
- Fixed positioning
